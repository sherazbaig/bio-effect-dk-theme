# Introduction

The Canvas Storybook is powered by [Storybook](https://storybook.js.org/), an open source tool for building UI components and pages in isolation. Storybook allows us to preview and interact with Canvas components outside of a Shopify store.

Due to the fact that it sits outside of a Shopify store only Vue templated components can be previewed in Storybook, static components can not be previewed. All Liquid supplied data must be manually passed to the components.

Please note that initial load will take a few seconds due to file sizes.
